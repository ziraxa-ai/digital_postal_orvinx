ZIRAXA – Interactive Digital Brochure (HTML)

How to use:
1) Open index.html in a modern browser (Chrome/Edge/Firefox).
2) Click the cover or the “باز کردن بروشور” button to play the open animation.
3) Navigate with:
   - ← / → arrow keys
   - Scroll inside the brochure
   - Click page dots
4) “خروجی PDF (پرینت)” uses your browser Print dialog (Save to PDF).

Customization:
- Edit content inside index.html (pages 1..5).
- Brand colors in styles.css:
  --bg, --gold, --gold2
- Replace logo: assets/ziraxa-logo.png

Tip:
For hosting, upload the folder to any static hosting (e.g. your website) and link to index.html.
